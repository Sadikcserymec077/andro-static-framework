About this Project

This is a demo Static Analysis Framework for Android applications that integrates with MobSF to run static scans of APK files and visualize results in a modern, responsive dashboard.

Features
  • Upload APK and trigger MobSF scan
  • View permissions, manifest issues, third-party libraries
  • Interactive charts and vulnerability table
  • Simple mitigation tips and export-ready JSON report
Future work
  • PDF report generation & email summary
  • Integrate additional tools: JADX, APKTool, QARK
  • Store history in MongoDB with user accounts
